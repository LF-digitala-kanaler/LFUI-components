 # Deprecated Components

 ## Charts
 Charts have been deprecated in v10 of LFUI. The following files need to be included manually to get the chart styling in your project. These files will be removed in future versions of LFUI.

 - src/scss/deprecated/**_chart.scss**
 - src/scss/deprecated/**_distribution-bar.scss**
 - src/scss/deprecated/**_pie-chart.scss**
