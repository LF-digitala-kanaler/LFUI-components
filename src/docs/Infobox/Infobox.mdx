## Guidelines

### Use an Infobox:
- To highlight short crucial information so it doesn’t disappear in a chunk of text.
- When the message needs to be permanently on the screen.
- When you need to tell something without shouting, just by raising your voice slightly.

### Don´t use an Infobox:
- If the message needs to be temporary (use alert instead)
- For error messages, then use another way (for examples helptexts, fullside error, alert...)
- For longer texts, information in an Infobox is short.
